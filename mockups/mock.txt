1. Dashboard Principal - Panel de Control del Laboratorio

Diseña un dashboard profesional de nivel empresarial para el Laboratorio Remoto Atriz con las siguientes especificaciones:

**DISEÑO GENERAL:**
- Layout moderno tipo "Enterprise Dashboard" con paleta de colores profesional
- Colores principales: Azul corporativo (#1e40af), Verde éxito (#10b981), Rojo alerta (#ef4444)
- Tipografía: Inter/Poppins para títulos, Roboto Mono para datos técnicos
- Espaciado consistente con grid de 8px
- Sombras sutiles y bordes redondeados (4px)

**HEADER SUPERIOR:**
- Barra superior fija con altura 64px
- Logo "Atriz Lab" a la izquierda con icono de robot
- Indicador de estado del sistema (punto verde/rojo + texto)
- Notificaciones (badge con número) + perfil de usuario (avatar + dropdown)
- Botón de emergencia rojo "STOP ALL" en esquina derecha

**SIDEBAR IZQUIERDO (240px de ancho):**
- Navegación vertical con iconos + texto
- Secciones: Dashboard, Experimentos, Robots, Telemetría, Video, Configuración
- Indicador activo con línea azul lateral
- Collapsible para pantallas pequeñas
- Footer con versión del sistema

**ÁREA PRINCIPAL - GRID 2x2:**

**Cuadrante Superior Izquierdo - "Estado de Robots":**
- Card con sombra sutil y padding 24px
- Título "Robots Activos" con icono de robot
- Grid de 3x2 robots con cards individuales:
  * Avatar circular con color de estado
  * Nombre del robot (RVR-001, RVR-002, etc.)
  * Estado: "Conectado" (verde) / "Desconectado" (rojo) / "Ocupado" (amarillo)
  * Barra de batería horizontal con porcentaje
  * Última actividad (timestamp relativo)
- Botón "Ver Todos" en esquina inferior derecha

**Cuadrante Superior Derecho - "Experimentos Activos":**
- Card con header azul y título "Experimentos en Ejecución"
- Lista vertical de experimentos:
  * Avatar del usuario + nombre
  * Nombre del experimento (truncado)
  * Robot asignado
  * Barra de progreso circular (0-100%)
  * Tiempo transcurrido (formato MM:SS)
  * Botones de acción: Pausar (amarillo), Detener (rojo)
- Indicador de experimentos en cola (badge numérico)

**Cuadrante Inferior Izquierdo - "Métricas del Sistema":**
- Card con título "Rendimiento del Sistema"
- Grid de métricas 2x2:
  * "Latencia Promedio": 45ms (verde si <50ms)
  * "Conexiones Activas": 12/50 (barra de progreso)
  * "CPU Servidor": 34% (barra de progreso)
  * "Memoria": 2.1GB/8GB (barra de progreso)
- Gráfica mini de tendencia (últimos 5 minutos)
- Botón "Ver Detalles" para métricas completas

**Cuadrante Inferior Derecho - "Video del Laboratorio":**
- Card con video player integrado
- Header con controles: Play/Pause, Grabación (botón rojo), Pantalla completa
- Overlay semitransparente con información:
  * Número de robots visibles
  * Latencia del video (ms)
  * Resolución actual
- Indicador de calidad de conexión (barras de señal)

**ELEMENTOS ADICIONALES:**
- Toast notifications en esquina superior derecha
- Modal de configuración rápida (accesible desde header)
- Indicador de estado WebSocket (punto verde/rojo)
- Loading states con skeletons
- Responsive design para tablets y móviles
